## Why this project?

**Diablo I Companion** aims to preserve and modernize community knowledge about the original **Diablo (1996)** and **Diablo: Hellfire** in an offline-first, open-source companion.

The project is available in two forms:

- **Web Edition:** runs in any modern browser and is also available through GitHub Pages.
- **Windows Edition:** a portable native desktop application for 64-bit Windows.

Neither edition requires an account, server connection or continuous internet access.

---

## Features

- Original Diablo Item Rarity Calculator
- Diablo / Hellfire mode
- English / Hungarian interface
- Dream Item Finder
- Holy Grail Tracker
- Base Item Encyclopedia
- Affix Encyclopedia
- Unique Monster Encyclopedia
- Wishlist
- Personal Notes
- JSON Export / Import
- Local save support
- Offline-first design

---

## Using the Web Edition

### Online

Open the application through GitHub Pages.

### Offline

Download `index.html` or the HTML package, then open it in any modern browser. No installation is required.

---

## Using the Windows Edition

Download the Windows ZIP from the repository's **Releases** section, extract it, then run:

```text
DiabloICompanion.exe
```

The Windows Edition is portable and does not require installation. It is intended for **64-bit Windows 10 and Windows 11**.

Windows may display a SmartScreen warning for a newly published unsigned open-source application. Review the downloaded file and choose **More info → Run anyway** only when it was downloaded from this repository's official Releases page.

---

## Privacy

No telemetry. No analytics. No accounts.

The Web Edition stores progress and settings locally in the browser. The Windows Edition stores its local state in the current Windows user's application-data folder and can export or import JSON backups.

No personal information is transmitted anywhere by the application.

# Magyar

## Mi ez?

A **Diablo I Companion** egy nyílt forráskódú, offline használatra tervezett segédprogram az eredeti **Diablo (1996)** és a **Diablo: Hellfire** játékokhoz.

A projekt kétféle formában érhető el:

- **Webes kiadás:** modern böngészőben fut, és GitHub Pagesen keresztül is használható.
- **Windows-kiadás:** hordozható, natív asztali alkalmazás 64 bites Windowshoz.

Egyik kiadás használatához sincs szükség fiókra, szerverkapcsolatra vagy folyamatos internetelérésre.

---

## Funkciók

- Eredeti Diablo Item Rarity Calculator
- Diablo / Hellfire mód
- Angol és magyar felület
- Álomtárgy-kereső
- Holy Grail követő
- Alaptárgy-enciklopédia
- Affixenciklopédia
- Unique szörny enciklopédia
- Kívánságlista
- Jegyzetek
- JSON export / import
- Helyi mentés
- Offline működés

---

## A webes kiadás használata

### Online

Nyisd meg az alkalmazást GitHub Pagesen keresztül.

### Offline

Töltsd le az `index.html` fájlt vagy a HTML-csomagot, majd nyisd meg egy modern böngészőben. Telepítés nem szükséges.

---

## A Windows-kiadás használata

Töltsd le a Windows ZIP-et a repository **Releases** részéből, csomagold ki, majd indítsd el ezt:

```text
DiabloICompanion.exe
```

A Windows-kiadás hordozható, telepítést nem igényel, és **64 bites Windows 10, illetve Windows 11** rendszerhez készült.

A Windows SmartScreen figyelmeztetést jeleníthet meg egy új, digitálisan nem aláírt nyílt forráskódú alkalmazásnál. A **További információ → Futtatás mindenképpen** lehetőséget csak akkor használd, ha a fájlt ennek a repositorynak a hivatalos Releases oldaláról töltötted le.

---

## Adatvédelem

Nincs telemetria. Nincs analitika. Nincs fiók.

A webes kiadás a böngésző helyi tárhelyén őrzi az adatokat. A Windows-kiadás az aktuális Windows-felhasználó alkalmazásadat-mappájában tárolja a helyi állapotot, és JSON-mentést tud exportálni, illetve importálni.

Az alkalmazás semmilyen személyes adatot nem továbbít.
