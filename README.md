## Why this project?

**Diablo I Companion** aims to preserve and modernize community
knowledge about the original **Diablo (1996)** and **Diablo: Hellfire**
in a single, offline-first, open-source application.

The goal is to provide collectors, theorycrafters and long-time fans
with a practical toolkit that works everywhere, without requiring an
installation, server or internet connection after download.

------------------------------------------------------------------------

## Features

-   Original Diablo Item Rarity Calculator
-   Diablo / Hellfire mode
-   English / Hungarian interface
-   Dream Item Finder
-   Holy Grail Tracker
-   Base Item Encyclopedia
-   Affix Encyclopedia
-   Unique Monster Encyclopedia
-   Wishlist
-   Personal Notes
-   JSON Export / Import
-   Local save support
-   Offline-first design

------------------------------------------------------------------------

## Running

### Online

Run directly from GitHub Pages.

### Offline

Download the latest HTML release and open it in any modern browser.

No installation required.

------------------------------------------------------------------------

## Privacy

Everything is stored locally inside your browser.

Stored locally:

-   Holy Grail progress
-   Notes
-   Wishlist
-   Selected language
-   Selected game

No telemetry.

No analytics.

No accounts.

No information is transmitted anywhere.

A **Diablo I Companion** egy nyílt forráskódú, offline használatra
tervezett segédprogram az eredeti **Diablo (1996)** és a **Diablo:
